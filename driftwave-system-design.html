Dual Vector Strategy
Two collections, merged at query time
📝
Text RAG Collection
384-dim · all-MiniLM-L6-v2
embeds: title + artist + composer + genre + tempo_label + mood_label + description + lyrics
strength: cultural context, lyrics meaning, composer knowledge
🎧
CLAP Audio Collection
512-dim · DCLAP (distilled)
embeds: raw audio waveform (30s)
strength: works with zero tags, pure sonic similarity, unknown language fallback
⚖️
Merge Formula
final_score = 0.6 × text_score + 0.4 × clap_score
weights tunable per language adapter
Tamil: 0.7/0.3 (tags reliable)
Unknown: 0.3/0.7 (CLAP dominant)
