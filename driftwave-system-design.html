Dual Vector Strategy
Two collections · merged at query time
📝
Text RAG Collection — 384-dim
all-MiniLM-L6-v2
embeds: title + artist + composer + genre + tempo_label + mood_label + description + lyrics
strength: cultural context, lyrics meaning, composer knowledge
🎧
CLAP Audio Collection — 512-dim
DCLAP (distilled) · raw waveform (30s)
strength: zero-tag files · pure sonic · unknown language fallback
⚖️
Merge Formula
final = 0.6 × text + 0.4 × CLAP
Tamil: 0.7/0.3 · Unknown: 0.3/0.7
Payload Indexes · Required for MatchText · New
TEXT INDEX (word tokenizer · MatchText)
cultural_meta.film_name cultural_meta.lyricist cultural_meta.composer cultural_meta.film_meta.cast artist lyrics
KEYWORD INDEX (MatchAny)
adapter_type year genre key cultural_meta.tamil_genre
